## Brawlbound

Brawlbound is a multiplayer real-time strategy game developed in Unity. 
The project supports online multiplayer gameplay using Unity Netcode for GameObjects, Unity Lobby, and Unity Relay.

Players control units, construct buildings, produce new units, research technological upgrades, and try to defeat opposing players. 
A player wins when all other players have been defeated.

## Repository
This repository contains the source code for the game Brawlbound in the src folder.

The documentation for the project can be found in the docs folder.

Built versions of the game that contain the executable file are located in the release section of the repository.

## Project Description

This project was created as part of a university project. 
The goal was to develop a functional base for a multiplayer RTS game, including systems for:

- creating and joining lobbies
- connecting players through Unity Relay
- synchronizing game objects over the network
- unit control
- building construction
- unit production
- combat between players
